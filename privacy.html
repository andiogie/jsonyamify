Kebijakan Privasi JSONYAMify

Terakhir diperbarui: 19 Juni 2026

JSONYAMify adalah tool gratis untuk memformat dan mengedit data JSON/YAML, yang berjalan 100% di browser kamu (client-side). Halaman ini menjelaskan data apa yang diproses dan apa yang tidak.

Data yang Diproses Tool

Semua data JSON/YAML yang kamu paste, upload, atau edit di JSONYAMify diproses sepenuhnya di browser kamu sendiri. Tidak ada data input yang dikirim, disimpan, atau diunggah ke server mana pun. Setelah kamu menutup atau me-refresh tab, seluruh data tersebut hilang — kami tidak punya salinannya.

Cookie & Analytics

Saat ini JSONYAMify tidak menggunakan cookie pelacakan maupun layanan analitik pihak ketiga. Jika di kemudian hari kami menambahkan analitik (misalnya untuk memahami fitur mana yang paling dipakai) atau iklan (misalnya Google AdSense), halaman ini akan diperbarui terlebih dahulu, dan layanan tersebut dapat menggunakan cookie sesuai kebijakan privasi masing-masing penyedia layanan (contoh: Google Ads & Cookies).

Hosting & Log Server

Situs ini di-hosting di layanan hosting statis (seperti Cloudflare Pages/GitHub Pages/Vercel). Penyedia hosting tersebut dapat secara otomatis mencatat log teknis standar (misalnya alamat IP, jenis browser, waktu akses) untuk keperluan keamanan dan performa infrastruktur — ini berada di luar kendali JSONYAMify dan diatur oleh kebijakan privasi penyedia hosting yang bersangkutan.

File yang Kamu Upload/Download

Fitur upload dan download file pada JSONYAMify bekerja langsung di browser (menggunakan File API bawaan browser). File yang kamu upload tidak pernah meninggalkan perangkatmu, dan file yang kamu download dibuat langsung di browser tanpa melalui server kami.

Saran Penggunaan

  • Karena JSONYAMify adalah tool publik, sebaiknya hindari menempelkan data yang sangat sensitif (kredensial, data pribadi/PII, data finansial atau perbankan) di komputer/perangkat bersama atau publik.
  • Meskipun pemrosesan dilakukan secara lokal di browser, gunakan tool ini secara bijak sesuai konteks kerahasiaan data kamu.
